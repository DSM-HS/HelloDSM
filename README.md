# Hello DSM
> Wecome to DSM!

This project is a project that outputs **"Hello DSM!"** In all programming languages.

# How to contribute
1. Add a file with a name like `dsm.js`

2. Write code that prints "Hello DSM!"
``` javascript
alert('Hello DSM!');
```

3. And just commit and push!
